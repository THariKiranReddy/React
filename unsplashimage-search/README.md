# Image Search App
  This is a Image search app which consists a search bar and theme toggle function and based on user search this shows the images.
## TechStack
   1.React
      i.useState.
      ii.useContext.
      iii.useQuery.
   2.TankStack and TankStackDevTools 
      * In this i have used TanStack Query Handles data fetching, caching, and synchronization.
      * TanStackDevTools is used for to know state of the component.
   3. Unsplash API
      * From here i have got the API key for data fetching.
